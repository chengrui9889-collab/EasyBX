## 技术债务 / 注意事项

1. **AC-007 浏览器端降级验证**：Zustand store 在 SPA 页面切换时保留缓存数据，停止后端后硬刷新时数据依然来自缓存。TDD 单元测试已覆盖此场景。生产环境中此行为合理（到期重新请求时才会触发 error）。
2. **port 2233 测试端口**：浏览器端验收时曾临时将 Vite proxy target 指向 2233（测试后端），已恢复回 220（开发后端）。理论上此操作不会对开发环境造成残留影响。
3. **前端 `vite.config.ts`**：proxy target 已确认恢复为 `http://localhost:220`。
